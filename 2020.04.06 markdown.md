### 글씨 강조

샾(#) 하나 -> 가장 큰 글씨  
샾(#) 여섯개 -> 가장 작은 글씨  
기울기 별 하나 -> It is *italic*  
볼드 별 두개 -> It is **bold**  
기울기+볼드 별 세개(***) -> It is ***italic and bold***  
취소선 물결 -> It is ~~cancelled~~
